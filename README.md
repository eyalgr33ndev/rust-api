<h1 align="center"><b>Examples</b></h1>

<p align="center">
<b>RustApi.NewServer</b> 

```
const { RustApi } = require('rust-api');

const server = RustApi.newServer('id');
```
</p>

<p align="center">
<b>getServerStatus()</b> 

```
const { RustApi } = require('rust-api');

const server = RustApi.newServer('id');


server.getServerStatus()
    .then(status => {
        console.log(status);
    })
    .catch(err => console.error(err));
```
</p>
